Air Quality India Analysis

Problem Statement: Rising air pollution in Indian cities is a major health crisis, but understanding long-term trends and seasonal triggers requires analyzing massive datasets.
Solution: Developed an interactive dashboard using Python and Power BI to visualize 10 years of AQI data, enabling users to track pollution hotspots and assess the impact of events like lockdowns.
Python Pandas NumPy Seaborn Plotly Power BI

Key Insights

  • Winter Smog: Delhi records highest PM2.5 levels (Nov-Jan).
  • Lockdown Impact: 40% drop in pollution during 2020.
  • Correlation: PM2.5 & PM10 are primary AQI drivers.

Solution: Analyzed the Netflix dataset to visualize content distribution by country, genre, and release year using Python.
Python Pandas Matplotlib Seaborn Power BI

Key Insights
